选择会议室

接待大会议室使用指南

容纳20-30人，配备高清腾讯极光投影、无线投屏等功能

位置

行政接待大厅，左手大会议室

容量

最多30人

设备

极光投影、内部视频系统、无级投屏、会议小电脑、音箱系统

使用步骤

1

打开电源

1.1 首先打开极光投影仪的电池开关，位于屏布墙最右侧最右边一个开关
1.2 再打开电视背后插板开关

2

开启会议室设备

使用墙上的控制面板开启投影仪、灯光和空调。控制面板位于入口右侧。

按下主电源按钮
调节灯光亮度
3

连接演示设备

使用HDMI或USB-C线缆连接您的笔记本电脑到会议桌上的接口。也可以使用无线投屏功能。

无线投屏步骤：1. 按遥控器上的"无线"按钮 2. 在设备上搜索并连接"会议室一"网络 3. 打开投屏应用并输入PIN码
4

使用视频会议系统

如需视频会议，请使用会议桌上的视频会议设备。按下电源按钮，然后按照屏幕提示操作。

拨打电话会议号码
加入视频会议
5

会议结束后

会议结束后，请清理桌面，关闭所有设备，并确认门锁好。

如发现设备故障，请在系统中报告或联系IT支持。

常见问题

投影仪的焦距调节旋钮位于设备的前部。顺时针旋转可缩小焦距，逆时针旋转可增大焦距。调整时请保持投影画面清晰。

1. 确认投影仪已开启并连接到网络
2. 检查您的设备是否能搜索到"会议室一"网络
3. 重启投影仪和您的设备
4. 如问题仍然存在，请联系IT支持

1. 检查视频会议设备的音量控制按钮
2. 确认麦克风没有被静音
3. 尝试调整会议桌上的音频控制面板
4. 如问题仍然存在，请联系IT支持

非接待大会议室使用指南（使用流程基本相同）

大小不一，配备4K电视显示、投影仪、内部会议系统、八爪鱼麦克风等

位置

接待大厅2、3会议室，装配二楼大小会议室

容量

10人

设备

4K电视、八爪鱼、投影仪、内部会议系统

使用步骤

1

打开设备电源

1.1 打开投影仪电源
1.2 如需使用电视，打开电视电源

2

调整正确的信号源

2.1 hdml1为公司内部会议系统信号源（监控显示+八爪鱼音频）
2.2 hdml2为小电脑的信号源
2.3 hdml3为外接笔记本或其它设备信号源

按下主电源按钮
调节灯光亮度
3

连接演示设备

使用HDMI线连接您的笔记本电脑（快捷键Ctrl+P,选）

无线投屏步骤：1. 按遥控器上的"无线"按钮 2. 在设备上搜索并连接"会议室二"网络 3. 打开投屏应用并输入PIN码
4

使用音响系统

如需使用音响系统，请按下控制面板上的"音频"按钮，然后调节音量。

增大音量
减小音量
5

会议结束后

会议结束后，请清理桌面，关闭所有设备，并确认门锁好。

如发现设备故障，请在系统中报告或联系IT支持。

常见问题

显示器支架上有调节旋钮，松开后可调整角度，调整后请紧固旋钮以保持稳定。

1. 确认显示器已开启并连接到网络
2. 检查您的设备是否能搜索到"会议室二"网络
3. 重启显示器和您的设备
4. 如问题仍然存在，请联系IT支持

1. 检查遥控板上的音量调节按钮
2. 确认设备和电脑没有被静音
3. 尝试调整设备上的音量设置
4. 如问题仍然存在，请联系IT支持
